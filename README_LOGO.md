# LOGMCP Logo 文件说明

本项目提供了多种格式的 logo，可根据需要选择使用。

## Logo 文件列表

### 1. logo-ascii.txt
**ASCII 艺术风格** - 适合在终端、README 中显示
```
  _                    _       _   _   _ 
 | |                  | |     | \ | | | |
 | |     ___  ___  ___| |__   |  \| | | |
 | |    / _ \/ __|/ __| '_ \  | . ` | | |
 | |___| (_) \__ \ (__| | | | | |\  | |_|
 |______\___/|___/\___|_| |_| |_| \_| (_)
                                          
  __  __  ____  ____  
 |  \/  |/ __ \|  _ \ 
 | \  / | |  | | |_) |
 | |\/| | |  | |  _ < 
 | |  | | |__| | |_) |
 |_|  |_|\____/|____/ 
                      
     MCP Log Analyzer
        Version 1.0.2
```

### 2. logo-minimal.svg
**简约 SVG 版本** - 适合在网页、文档中使用
- 蓝色渐变背景
- 简洁的设计
- 可缩放矢量图

### 3. logo.svg
**完整 SVG 版本** - 带装饰元素
- 深蓝色背景
- 装饰性边框和线条
- 更丰富的视觉效果

### 4. logo.txt / logo-simple.txt
**文本边框版本** - 适合在文档中显示

## 使用建议

- **README.md**: 使用 `logo-ascii.txt` 或 `logo-minimal.svg`
- **PyPI 项目页面**: 使用 `logo-minimal.svg`
- **文档**: 使用 `logo-ascii.txt`
- **网站**: 使用 `logo.svg` 或 `logo-minimal.svg`

## 在 README 中使用

### 方式 1：ASCII 艺术
```markdown
![LOGMCP Logo](logo-ascii.txt)
```

### 方式 2：SVG 图片
```markdown
![LOGMCP Logo](logo-minimal.svg)
```

### 方式 3：直接嵌入 ASCII
复制 `logo-ascii.txt` 的内容到 README.md 中。
