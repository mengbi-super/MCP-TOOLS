# PyPI 上传说明

## 已完成的步骤

✅ 1. 安装构建工具（build 和 twine）
✅ 2. 清理旧的构建文件
✅ 3. 构建包（已生成 dist/ 目录下的文件）
✅ 4. 检查包（通过验证）

## 上传到 PyPI

### 方式一：使用命令行（推荐）

运行以下命令，然后按提示输入：

```bash
py -m twine upload dist/*
```

**输入信息：**
- Username: `__token__`
- Password: `你的 PyPI token`（粘贴你的 token，不会显示）

### 方式二：使用环境变量

```bash
# Windows PowerShell
$env:TWINE_USERNAME="__token__"
$env:TWINE_PASSWORD="你的token"
py -m twine upload dist/*

# Windows CMD
set TWINE_USERNAME=__token__
set TWINE_PASSWORD=你的token
py -m twine upload dist/*
```

### 方式三：使用配置文件

创建或编辑 `%USERPROFILE%\.pypirc` 文件：

```ini
[pypi]
username = __token__
password = 你的token
```

然后运行：
```bash
py -m twine upload dist/*
```

## 上传后的验证

上传成功后，可以：

1. 访问 https://pypi.org/project/mcp-logback-analyzer/ 查看你的包
2. 测试安装：
   ```bash
   pip install mcp-logback-analyzer
   ```

## 注意事项

- Token 格式：`pypi-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
- 确保 token 有上传权限
- 如果包名已存在，需要更新版本号
- 上传后无法删除，只能发布新版本
